# Fuel Consumption Calculator with Multi-Language Support

This JavaFX application calculates fuel consumption in liters per 100 kilometers.

## Features
- Enter distance and fuel amount
- Calculate fuel consumption
- Multi-language support:
  - English
  - French
  - Japanese
  - Persian

## Formula
Fuel Consumption = (Fuel / Distance) × 100

## Example
Distance = 100 km  
Fuel = 8 L  
Result = 8.00 L/100 km

## How to Run
```bash
mvn clean javafx:run
